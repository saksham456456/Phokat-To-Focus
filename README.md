# Phokat-To-Focus 🎯

**Phokat-To-Focus** is a modern, AI-powered study companion designed to help students transition from "Phokat" (distracted/unproductive) time to deep, focused study sessions.

## 🚀 Features

- **AI-Powered Planner:** Automatically generates study schedules based on your goals and deadlines.
- **Strict Mode (Android):** Prevents you from leaving the app during focus sessions using system-level overlays and re-launching logic.
- **AI Coach:** Chat with an AI mentor to get tips on complex topics or motivation when you're feeling stuck.
- **Progress Tracking:** Beautiful charts to visualize your study habits and improvements.
- **Cross-Platform:** Available on Android and Web.

## 🛠️ Tech Stack

- **Framework:** Flutter
- **State Management:** Provider
- **Backend:** Firebase (Auth, Firestore)
- **Database:** Local Storage (Shared Preferences)
- **AI Integration:** Google Generative AI (Gemini)

## 📦 Getting Started

### Prerequisites

- Flutter SDK (latest stable)
- Android Studio / VS Code
- Firebase Project (for sync features)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Phokat-To-Focus.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Phokat-To-Focus
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## 📱 Platforms Supported

- ✅ Android
- ✅ Web (PWA)
